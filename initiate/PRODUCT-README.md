# Peptide Portal — Product Reference

*For sales conversations*

---

## What It Is

A patient acquisition and management platform for peptide therapy providers.

**Who it's for:** Physicians in Utah offering peptide treatments (weight loss, recovery, longevity, cognitive)

**What it does:** Automates patient intake, enables recurring subscriptions, handles compliance

---

## Features

### For Patients
- [ ] Online health assessment (quiz)
- [ ] Personalized protocol recommendations
- [ ] Patient dashboard with protocol tracking
- [ ] Secure messaging with provider
- [ ] Subscription management
- [ ] Order tracking

### For Providers
- [ ] Provider dashboard with intake queue
- [ ] One-click approve/deny prescriptions
- [ ] Patient management (up to 100)
- [ ] Secure messaging
- [ ] Analytics dashboard
- [ ] HIPAA-compliant
