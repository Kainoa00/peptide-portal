## 📞 Outreach Workflow

### Step 1: Research (5 min per lead)
- [ ] Find clinic website
- [ ] Find doctor name(s)
- [ ] Find phone/email
- [ ] Note specializations
- [ ] Add to CRM

### Step 2: First Contact (Day 0)
- [ ] Send email (use template from SALES-OUTREACH.md)
- [ ] Log in CRM with status "Contacted"
- [ ] Set follow-up for Day 3

### Step 3: Follow-Up (Day 3)
- [ ] Send follow-up email
- [ ] If no response → Phone call
- [ ] Log in CRM

### Step 4: Book Demo (Day 7-14)
- [ ] Offer 10-min demo
- [ ] Send calendar link
- [ ] Set status "Demo Scheduled"

### Step 5: Demo & Close
- [ ] Run demo (use DEMO-SCRIPT.md)
- [ ] Send follow-up with pricing
- [ ] Handle objections
- [ ] Close: "Should I set you up on the free pilot?"
